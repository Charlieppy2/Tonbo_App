# 瞳伴 (Tonbo) - 智能視覺助手

瞳伴是一個專為視障人士設計的智能助手應用，結合計算機視覺技術和語音提示，幫助用戶在日常生活中進行導航和環境感知。

## 🎯 核心功能

### 🏠 首頁與導航 ✅ **已完成**
- **語音歡迎**: 應用啟動時的語音引導和功能介紹
- **快速啟動面板**: 四大核心功能一鍵訪問
- **多語言支持**: 廣東話、英文、普通話三語言切換
- **緊急求助**: 長按3秒觸發緊急警報系統
- **震動反饋**: 觸控和操作反饋

### 🌍 環境感知 ✅ **已完成**
- **實時物體識別**: 基於TensorFlow Lite的YOLO模型檢測
- **檢測框顯示**: 實時在相機預覽上顯示檢測結果
- **顏色分析**: 主要顏色、次要顏色、色調分析
- **光度檢測**: 亮度等級、對比度、光線方向分析
- **環境描述**: 綜合物體、顏色、光線的語音描述
- **實時語音播報**: 檢測結果的即時語音反饋

### 📖 文件與貨幣閱讀 ✅ **已完成**
- **OCR文字識別**: 基於Google ML Kit的中文文字識別
- **港幣識別**: 專門的港幣紙幣和硬幣識別系統
- **分離式功能**: 文字識別和貨幣識別獨立操作
- **掃描區域指示**: 320×240dp大範圍掃描框
- **語音朗讀**: 識別結果的語音播報
- **方形按鈕設計**: 優化的無障礙按鈕布局

### ⚠️ 障礙物偵測 🔄 **規劃中**
- **實時障礙物警告**: 檢測前方障礙物
- **樓梯偵測**: 識別樓梯和台階
- **安全導航**: 語音導航提示

### 🔍 尋找功能 🔄 **規劃中**
- **標記個人物品**: 為重要物品添加標記
- **智能尋找**: 使用攝像頭尋找標記物品
- **位置記憶**: 記住物品位置

### 🆘 即時援助 🔄 **規劃中**
- **一鍵呼叫**: 快速聯繫志願者
- **快速訊息**: 預設緊急訊息
- **緊急求助**: 長按3秒觸發緊急警報

### 👤 用戶系統 🔄 **規劃中**
- **登入/註冊**: 用戶賬戶管理
- **個人化設定**: 自定義設置同步
- **偏好設定**: 語音和界面偏好

### 📚 引導與簡介 ✅ **已完成**
- **啟動畫面**: 帶動畫的應用啟動頁面
- **無障礙設計**: 專為視障用戶優化的UI/UX
- **語音導航**: 每個頁面的語音引導

### ⚙️ 系統設定 ✅ **部分完成**

- **震動管理**: 觸控反饋系統

## 🛠️ 技術架構

### **核心組件**
- `TTSManager`: 多語言語音合成管理
- `VibrationManager`: 震動反饋管理
- `BaseAccessibleActivity`: 無障礙基礎活動
- `ObjectDetectorHelper`: TensorFlow Lite物體檢測
- `ColorLightingAnalyzer`: 顏色和光度分析
- `OCRHelper`: Google ML Kit文字識別
- `CurrencyDetector`: 港幣識別系統
- `DetectionOverlayView`: 檢測框覆蓋層

### **主要頁面**
- `MainActivity`: 首頁功能選擇
- `EnvironmentActivity`: 環境識別和實時檢測
- `DocumentCurrencyActivity`: 文件和貨幣閱讀
- `SplashActivity`: 啟動畫面

## 🔧 安裝與使用

### 編譯要求
- Android Studio Arctic Fox 或更新版本
- Android SDK API 21+
- Gradle 8.0+

### 編譯步驟
```bash
git clone https://github.com/Charlieppy2/Tonbo_App.git
cd Tonbo_App/Tonbo_App/Tonbo_App
./gradlew assembleDebug
```

### 使用指南
1. 安裝APK到Android設備
2. 授予攝像頭、麥克風、震動權限
3. 啟動應用，聽取語音引導
4. 使用語音導航操作各功能
