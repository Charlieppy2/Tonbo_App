## 🤝 如何成為貢獻者

歡迎更多開發者參與瞳伴項目的開發！特別是以下領域的專家：

- **計算機視覺算法**: 物體檢測、OCR優化
- **無障礙技術**: 視障用戶體驗優化
- **Android開發**: 性能優化、UI/UX改進
- **語音合成技術**: TTS質量提升

### 貢獻指南
1. Fork 本項目
2. 創建功能分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 開啟 Pull Request
